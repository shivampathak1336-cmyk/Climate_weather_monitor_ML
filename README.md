# Climate_weather_monitor_ML

Climate change has caused increasingly erratic temperature patterns, posing serious threats to agriculture, public health, and disaster preparedness. Conventional forecasting systems struggle to account for the long-term climate shifts driven by global warming, making accurate predictions unreliable. This project addresses the need for a robust, data-driven solution that leverages historical climate records and machine learning to forecast future temperatures — enabling better weather prediction, informed decision-making, and proactive climate resilience.
